Lightgrid

Version 1.0.3
Latest version published on 3/23/13

A simple, fluidic and responsive 12 column grid system for css.

Lightgrid css will divide your page into 12 columns. You can arrange the divs to take up column space by adding col-* classes.
One thing: Don't forget to add the .last class in your last div.


Try it now

First, add lightgrid.css in your html file.
Add <div class="row"></div> in your file.
Example:
<div class="row">
<div class="col-5">
col-5
</div>
<div class="col-5">
col-5
<div class="col-2 last">
col-2 last
</div>
</div>

Extras


Offsetting

You can also offset a div by any number of columns.
You can do this by adding offset-* in your div class.

Example:
<div class="row">
<div class="col-4 offset-4">
col-4 offset-4
</div>
<div class="col-2 offset-2 last">
col-2 offset-2
</div>
</div>

col-4 offset-4
col-2 offset-2 last

Images

Images are responsive and resize according to the parent element's size.

col-6 last

Responsive

At 768px (ipad width), lightgrid will automatically make all divs to 100% width.
Go ahead, resize this window to see it in action.


Trimming

If you are sure you don't need to change the layout of your website, remove the extra classes in lightgrid.css. This will remove the clutter from your css file.
Also, take a look under the hood and delete the sections you don't need in your site. For example, if you are already using a reset or normalize css then remove the first section of lightgrid.css


Support

Lightgrid is fully supported in Ie7 and above.
